Задача 18: касательная к окружности из пучка прямых

Пошаговое решение с графическим методом

Задача. При каких значениях параметра $a$ прямая $y = ax + 1$ является касательной к окружности $x^2 + y^2 = 1$?

Рисуем единичную окружность $x^2 + y^2 = 1$ с центром в начале координат.

Окружность x² + y² = 1

Все прямые $y = ax + 1$ проходят через точку $(0; 1)$ — это пучок прямых.


Показаны прямые при $a = -1, 0, 1$ (штриховые).

Пучок прямых через (0; 1)

Условие касания: расстояние от центра $(0;0)$ до прямой $ax - y + 1 = 0$ равно $r = 1$:


$$\frac{|0 \cdot a - 0 + 1|}{\sqrt{a^2 + 1}} = 1$$

$$1 = \sqrt{a^2 + 1}$$
$$1 = a^2 + 1$$
$$a^2 = 0$$
$$a = 0$$

Ответ: $a = 0$. Прямая $y = 1$ — единственная касательная из пучка.

Касательная y = 1 при a = 0
