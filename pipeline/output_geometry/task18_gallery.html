Задача 18: x² − 2|x| = a — ровно 3 решения

Задача. При каких значениях параметра $a$ уравнение $x^2 - 2|x| = a$ имеет ровно 3 решения?

Перепишем уравнение: $y = x^2 - 2|x|$. Построим график этой функции.


При $x \geq 0$: $y = x^2 - 2x = (x-1)^2 - 1$, минимум $-1$ при $x=1$.

Функция чётная, поэтому график симметричен.

График y = x² − 2|x|

Критические значения параметра:

• $a = -1$: прямая $y = a$ касается минимумов (2 точки касания)
• $a = 0$: прямая проходит через начало координат
Критические уровни a = −1 и a = 0

Считаем число пересечений прямой $y = a$ с графиком:

• $a > 0$: 4 пересечения → **4 решения**
• $a = 0$: 3 пересечения (в т.ч. $x=0$) → **3 решения** ✓
• $-1 < a < 0$: 4 пересечения → **4 решения**
• $a = -1$: 2 пересечения (касание) → **2 решения**
• $a < -1$: 0 пересечений → **0 решений**

Ответ: $a = 0$.

Задача 18: |x² − 4| = a — ровно 2 решения

Задача. При каких значениях параметра $a$ уравнение $|x^2 - 4| = a$ имеет ровно 2 решения?

Строим график $y = |x^2 - 4|$.


Это парабола $x^2 - 4$, «отражённая вверх» ниже оси $OX$.

Нули: $x = \pm 2$. Вершина исходной параболы: $(0; -4) \to (0; 4)$ после отражения.

y = |x² − 4|

Критические значения:

• $a = 4$: максимум функции (прямая касается вершины)
• $a = 0$: прямая проходит через нули
Критические уровни a = 0 и a = 4

Число пересечений горизонтальной прямой $y = a$ с графиком:

• $a > 4$: **2 решения** ✓
• $a = 4$: **3 решения** (2 + касание)
• $0 < a < 4$: **4 решения**
• $a = 0$: **2 решения** ✓ (только $x = \pm 2$)
• $a < 0$: **0 решений** (правая часть отрицательна)

Ответ: $a > 4$ или $a = 0$.

Задача 18: касательная к окружности из пучка прямых

Задача. При каких значениях параметра $a$ прямая $y = ax + 1$ является касательной к окружности $x^2 + y^2 = 1$?

Рисуем единичную окружность $x^2 + y^2 = 1$ с центром в начале координат.

Окружность x² + y² = 1

Все прямые $y = ax + 1$ проходят через точку $(0; 1)$ — это пучок прямых.


Показаны прямые при $a = -1, 0, 1$ (штриховые).

Пучок прямых через (0; 1)

Условие касания: расстояние от центра $(0;0)$ до прямой $ax - y + 1 = 0$ равно $r = 1$:


$$\frac{|0 \cdot a - 0 + 1|}{\sqrt{a^2 + 1}} = 1$$

$$1 = \sqrt{a^2 + 1}$$
$$1 = a^2 + 1$$
$$a^2 = 0$$
$$a = 0$$

Ответ: $a = 0$. Прямая $y = 1$ — единственная касательная из пучка.

Касательная y = 1 при a = 0
Задача 18: x² − 2ax + a ≤ 0 — метод областей

Задача. При каких значениях параметра $a$ неравенство $x^2 - 2ax + a \leq 0$ имеет хотя бы одно решение?

Перепишем: $x^2 \leq 2ax - a$, то есть $x^2 \leq a(2x - 1)$.


При $2x - 1 > 0$ (т.е. $x > 1/2$): $a \geq \frac{x^2}{2x-1}$.


Рассмотрим кривую $a = \frac{x^2}{2x-1}$ на плоскости $(x, a)$.

Граничная кривая a = x²/(2x−1)

Находим минимум $a(x) = \frac{x^2}{2x-1}$ при $x > 1/2$:


$$a'(x) = \frac{2x(2x-1) - 2x^2}{(2x-1)^2} = \frac{2x^2 - 2x}{(2x-1)^2} = 0$$

$x(x-1) = 0$, при $x > 1/2$: $x = 1$. Тогда $a(1) = \frac{1}{1} = 1$.

Минимум граничной кривой при x = 1, a = 1

Неравенство имеет решение, когда прямая $y = a$ проходит через закрашенную область (выше граничной кривой).


Минимум кривой $= 1$, значит при $a \geq 1$ неравенство имеет решения.


Отдельно: при $a \leq 0$ также есть решения (при $x \leq 0$: $x^2 + a \leq 0$).


Ответ: $a \leq 0$ или $a \geq 1$.

Ответ: a ≤ 0 или a ≥ 1
