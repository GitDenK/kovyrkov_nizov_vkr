Задача 18: x² − 2ax + a ≤ 0 — метод областей

Пошаговое решение с графическим методом

Задача. При каких значениях параметра $a$ неравенство $x^2 - 2ax + a \leq 0$ имеет хотя бы одно решение?

Перепишем: $x^2 \leq 2ax - a$, то есть $x^2 \leq a(2x - 1)$.


При $2x - 1 > 0$ (т.е. $x > 1/2$): $a \geq \frac{x^2}{2x-1}$.


Рассмотрим кривую $a = \frac{x^2}{2x-1}$ на плоскости $(x, a)$.

Граничная кривая a = x²/(2x−1)

Находим минимум $a(x) = \frac{x^2}{2x-1}$ при $x > 1/2$:


$$a'(x) = \frac{2x(2x-1) - 2x^2}{(2x-1)^2} = \frac{2x^2 - 2x}{(2x-1)^2} = 0$$

$x(x-1) = 0$, при $x > 1/2$: $x = 1$. Тогда $a(1) = \frac{1}{1} = 1$.

Минимум граничной кривой при x = 1, a = 1

Неравенство имеет решение, когда прямая $y = a$ проходит через закрашенную область (выше граничной кривой).


Минимум кривой $= 1$, значит при $a \geq 1$ неравенство имеет решения.


Отдельно: при $a \leq 0$ также есть решения (при $x \leq 0$: $x^2 + a \leq 0$).


Ответ: $a \leq 0$ или $a \geq 1$.

Ответ: a ≤ 0 или a ≥ 1
