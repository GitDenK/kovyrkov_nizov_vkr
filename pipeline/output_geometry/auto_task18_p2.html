|x² − 4x + 3| = a — ровно 3 решения

Пошаговое решение с графическим методом

Строим график $y = |x^2 - 4x + 3|$.

Критические уровни параметра:

$a = 0$ — нули функции, $a = 1$ — локальный максимум «отражённой» части.

Считаем пересечения $y = a$ с графиком:

• $a < 0$: **0 решений**
• $a = 0$: **2 решения**
• $0 < a < 1$: **4 решения**
• $a = 1$: **3 решения**
• $a > 1$: **2 решения**

Ответ: $a = 1$.
