Задание 18 Профильного ЕГЭ по математике.

Найдите все значения параметра a, при каждом из которых уравнение
9^x − 4·3^x + a = 0
имеет ровно два решения.

Решение:
Замена t = 3^x, t > 0. Тогда t² − 4t + a = 0, то есть a = −t² + 4t.
Обозначим f(t) = −t² + 4t = −(t − 2)² + 4, t > 0.
Это парабола ветвями вниз с вершиной (2; 4).
f(t) → −∞ при t → +∞, f(0) = 0 (не включаем, t > 0).

Каждому t > 0 соответствует ровно одно x = log₃(t).
Значит, число решений по x = число корней t > 0 уравнения f(t) = a.

Горизонтальная прямая y = a:
- a < 0: 1 пересечение (правая ветвь) → 1 решение
- a = 0: 1 пересечение (при t = 4, т.к. t = 0 не входит) → 1 решение
- 0 < a < 4: 2 пересечения → 2 решения ✓
- a = 4: 1 пересечение (вершина, касание) → 1 решение
- a > 4: 0 пересечений → 0 решений

Ответ: 0 < a < 4.
