### 2. ТЕОРЕТИЧЕСКАЯ ЧАСТЬ

#### 🔹 Показательные уравнения

Уравнение вида $ a^{f(x)} = a^{g(x)} $, где $ a > 0 $, $ a \ne 1 $

📌 **Правило**: если основания равны и $ a > 0 $, $ a \ne 1 $, то  
$ a^{f(x)} = a^{g(x)} \Rightarrow f(x) = g(x) $

📌 Пример: $ 2^{x+1} = 2^3 \Rightarrow x+1 = 3 \Rightarrow x = 2 $

📌 Важно: если основание $ a = 1 $ — уравнение может иметь бесконечно много решений или не иметь, если основания разные.

---

#### 🔹 Логарифмические уравнения

Уравнение вида $ \log_a(f(x)) = \log_a(g(x)) $, где $ a > 0 $, $ a \ne 1 $

📌 **Правило**: если основания одинаковы, то  
$ \log_a(f(x)) = \log_a(g(x)) \Rightarrow f(x) = g(x) $, **при условии, что оба выражения положительны**

📌 Определение логарифма:  
$ \log_a(b) = c \Rightarrow a^c = b $, где $ a > 0 $, $ a \ne 1 $, $ b > 0 $

📌 ОДЗ для логарифма:  
- основание $ a > 0 $, $ a \ne 1 $  
- подлогарифмическое выражение $ > 0 $

---

#### 🔹 Дробно-рациональные уравнения

Уравнение вида $ \frac{P(x)}{Q(x)} = 0 $, где $ P(x) $, $ Q(x) $ — многочлены

📌 Правило: дробь равна нулю, когда числитель равен нулю, а знаменатель — **не равен нулю**

📌 То есть:  
$ \frac{P(x)}{Q(x)} = 0 \Rightarrow P(x) = 0 $ и $ Q(x) \ne 0 $

📌 Обязательно проверяй, чтобы знаменатель не обращался в ноль!

---

#### 🔹 Иррациональные уравнения

Уравнение, в котором переменная под корнем, например: $ \sqrt{f(x)} = g(x) $

📌 Правило: чтобы решить, **возведи обе части в квадрат**, но помни:

- $ \sqrt{f(x)} \ge 0 $, значит правая часть тоже должна быть $ \ge 0 $
- После возведения могут появиться **посторонние корни** — их нужно проверять!

📌 Пример: $ \sqrt{x} = 3 \Rightarrow x = 9 $ — проверим: $ \sqrt{9} = 3 $ — верно.

### 3. ТИПЫ ЗАДАЧ КОТОРЫЕ МОГУТ ПОПАСТЬСЯ

| Тип | Описание | Особенности |
|-----|---------|------------|
| **Показательное** | Уравнение со степенями, например $ 2^x = 8 $ | Приводи к одинаковому основанию |
| **Логарифмическое** | Уравнение с логарифмами, например $ \log_2(x) = 3 $ | Не забывай ОДЗ! |
| **Дробно-рациональное** | Дробь равна нулю или другому числу, например $ \frac{x+1}{x-2} = 0 $ | Числитель = 0, знаменатель ≠ 0 |
| **Иррациональное** | Переменная под корнем, например $ \sqrt{x+1} = 2 $ | Возведи в квадрат, проверь ответ |

### 4. АЛГОРИТМ РЕШЕНИЯ

#### 🔸 Для всех типов уравнений — общий план:

1. **Определи тип уравнения** — по наличию степени, логарифма, дроби, корня.
2. **Найди ОДЗ** (особенно важно для логарифмов и корней).
3. **Приведи уравнение к простому виду**:
   - Для показательных: сделай основания одинаковыми.
   - Для логарифмических: используй определение или приравнивание.
   - Для дробей: числитель = 0, знаменатель ≠ 0.
   - Для корней: возведи в квадрат, но помни про ограничения.
4. **Реши полученное уравнение** (обычно линейное или квадратное).
5. **Проверь корни по ОДЗ** — отбрось те, что не подходят.
6. **Запиши ответ** — целое число или десятичная дробь.

📌 **Частые ошибки**:
- Забыл про ОДЗ — и получил посторонний корень.
- Возвел в квадрат, не проверив знак правой части.
- Не проверил, не обращается ли знаменатель в ноль.
- Ошибся в преобразованиях степеней.

### 5. ПРИМЕРЫ РЕШЕНИЯ

#### ✏️ Пример 1: Показательное уравнение

**Задача**: Решите уравнение $ 4^{x+1} = 8 $

**Шаг 1**: Определи тип — показательное.

**Шаг 2**: Приведи к одинаковому основанию.  
Заметим: $ 4 = 2^2 $, $ 8 = 2^3 $  
Тогда:  
$ (2^2)^{x+1} = 2^3 $  
$ 2^{2(x+1)} = 2^3 $ → $ 2^{2x+2} = 2^3 $

**Шаг 3**: Приравниваем показатели:  
$ 2x + 2 = 3 $  
$ 2x = 1 $  
$ x = 0.5 $

**Шаг 4**: Проверяем — здесь ОДЗ нет ограничений (степень определена при любом $ x $), так что всё ок.

✅ Ответ: $ 0.5 $

---

#### ✏️ Пример 2: Логарифмическое уравнение

**Задача**: Решите уравнение $ \log_3(x+2) = 2 $

**Шаг 1**: Это логарифмическое уравнение.

**Шаг 2**: Найдем ОДЗ:  
- основание $ 3 > 0 $, $ 3 \ne 1 $ — нормально  
- подлогарифмическое выражение: $ x+2 > 0 \Rightarrow x > -2 $

**Шаг 3**: Используем определение логарифма:  
$ \log_3(x+2) = 2 \Rightarrow 3^2 = x+2 \Rightarrow 9 = x+2 \Rightarrow x = 7 $

**Шаг 4**: Проверим ОДЗ: $ x = 7 > -2 $ — подходит.

✅ Ответ: $ 7 $

---

#### ✏️ Пример 3: Дробно-рациональное уравнение

**Задача**: Решите уравнение $ \frac{x+1}{x-3} = 0 $

**Шаг 1**: Это дробь, равна нулю.

**Шаг 2**: Числитель = 0: $ x+1 = 0 \Rightarrow x = -1 $

**Шаг 3**: Знаменатель ≠ 0: $ x-3 \ne 0 \Rightarrow x \ne 3 $

**Шаг 4**: Проверяем: $ x = -1 \ne 3 $ — подходит.

✅ Ответ: $ -1 $

---

#### ✏️ Пример 4: Иррациональное уравнение

**Задача**: Решите уравнение $ \sqrt{x+3} = 2 $

**Шаг 1**: Переменная под корнем — иррациональное уравнение.

**Шаг 2**: ОДЗ: $ x+3 \ge 0 \Rightarrow x \ge -3 $

**Шаг 3**: Возведём обе части в квадрат:  
$ (\sqrt{x+3})^2 = 2^2 \Rightarrow x+3 = 4 \Rightarrow x = 1 $

**Шаг 4**: Проверим ОДЗ: $ x = 1 \ge -3 $ — подходит.  
Проверим в исходном уравнении: $ \sqrt{1+3} = \sqrt{4} = 2 $ — верно.

✅ Ответ: $ 1 $

### 6. ПРАКТИЧЕСКИЕ СОВЕТЫ

✅ **Как быстро решать**:
- Всегда сначала определяй тип уравнения.
- Ищи возможность привести к одинаковому основанию (для степеней).
- Для логарифмов — сразу пиши ОДЗ.
- Для дробей — помни: числитель = 0, знаменатель ≠ 0.
- Для корней — возводи в квадрат, но проверяй!

✅ **Как не ошибаться**:
- Не забывай про ОДЗ — особенно в логарифмах и корнях.
- После возведения в квадрат — **всегда проверяй корни**.
- Если в ответе два корня — читай условие: нужен ли наименьший, наибольший или все?

✅ **Запоминай**:
- $ a^0 = 1 $
- $ \sqrt{a^2} = |a| $ — но в уравнении типа $ \sqrt{x} = b $ — $ x = b^2 $, и $ b \ge 0 $
- $ \log_a(1) = 0 $, $ \log_a(a) = 1 $
